# Construction Management and Monitoring System

A full-stack Construction Management and Monitoring System designed to streamline project workflows, workforce coordination, and resource management across construction projects.

## Overview

This system provides an integrated platform for managing construction operations, including task assignment, attendance tracking, payroll processing, and project monitoring. It supports multiple user roles to ensure structured access and efficient collaboration.

## Features

- **Multi-Role Access Control**  
  Supports multiple user types with role-based permissions for secure and organized system usage.

- **QR-Based Attendance Tracking**  
  Workers can log attendance באמצעות QR code scanning for fast and accurate time tracking.

- **Task Management & Scheduling**  
  Create, assign, and monitor tasks with an integrated calendar view.

- **Project & Workforce Management**  
  Manage workers, roles, and project assignments efficiently.

- **Resource Management**  
  Track materials, quantities, and associated costs for each project.

- **Payroll System**  
  Automated payroll processing with support for bonuses and deductions.

- **Reports Generation**  
  Generate overall and weekly reports for project monitoring and decision-making.
